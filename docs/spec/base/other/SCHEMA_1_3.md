# Form

## 目的・用途

他のウェブアプリがweko3のリソースにアクセスできるようAPI利用を承認することを目的としている。

## 利用方法

API-8-5の機能を用いて、OAuthアプリケーション、またはトークンを登録する。その後、設定された値を利用してAPI接続の設定を行う。

## 機能内容

/items/schemaform/{item type id}

## 関連モジュール

- Invenio_oaiserver
