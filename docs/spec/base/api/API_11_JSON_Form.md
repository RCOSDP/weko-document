
### JSON Form

  - 目的・用途

インデックスIDを指定しての検索機能を提供する。

  - 利用方法

| **Method** | **HTTP request**                         | **Description** |
| ---------- | ---------------------------------------- | --------------- |
|            | **GET /items/schemaform/**{ITEMTYPE\_ID} | アイテムを検索する       |

パスパラメータ

| **GET /items/schemaform/**{ITEMTYPE\_ID} |     |             |
| ---------------------------------------- | --- | ----------- |
| パラメータ                                    | 値   | 説明          |
| ITEMTYPE\_ID                             | int | インデックスIDを指定 |
